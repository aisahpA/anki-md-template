
## 使用说明
直接导入牌组：`Markdown模板导入专用-离线模式.apkg`，即可使用


## 离线模式说明
离线模式只是将需要的 js 和 css 文件存放在媒体库中，不用远程获取
本地 JS 文件
```js
    jsResources : [
        "_highlight.min.js",
        "_markdown-it.min.js",
        "_mermaid.min.js"
    ]
```
本地 CSS 文件
```css
@import url('_github-markdown-css.min.css');
@import url('_highlight.min.css');
@import url('_normalize.min.css');
```
